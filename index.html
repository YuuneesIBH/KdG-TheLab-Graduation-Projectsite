◈ PROJECT

The Arcaders is een interactief arcade-kabinet gebouwd als Electron-applicatie. Het combineert een retro arcade-interface met een ingebouwde "Guppy" hacker-terminal — een tool voor NFC-cloning, IR-blasting en Wi-Fi deauth, verpakt in een nostalgisch CRT-jasje.

Spelers kiezen klassieke games via een draaiend selectiewiel. Op de achtergrond draait de Guppy-module als volledig uitgerust pentesting-dashboard, inclusief matrix-animaties, real-time seriële output en gamepad-navigatie.

De UI is gebouwd in React + TypeScript met Electron als native runtime — game UX meets security tooling.
