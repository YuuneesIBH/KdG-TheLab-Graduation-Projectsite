◈ OVERVIEW

The Arcaders is a retro arcade cabinet built as a conference project that mixes game nostalgia with custom hardware interactions.

The experience is split into two sides: a playable arcade frontend and the Guppy terminal, a hardware dashboard that exposes NFC, IR and wireless experiments in the same visual language.
